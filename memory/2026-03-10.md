## 3:22 PM - Root Cause: JwtAuthGuard Dependency Injection Conflict

- Original `JwtAuthGuard` extended `AuthGuard('jwt')` and injected `JwtService`
- This created a conflict: NestJS already provided `JwtService` via `AuthGuard`, but `JwtAuthGuard` tried to inject another instance
- Result: `this.jwtService` was undefined → failed to verify token

## 3:23 PM - Fix: Rewrite JwtAuthGuard as Independent Guard

- Removed inheritance from `AuthGuard('jwt')`
- Implemented standalone guard using direct `JwtService` injection
- Replaced return values with `UnauthorizedException` to ensure proper HTTP error responses
